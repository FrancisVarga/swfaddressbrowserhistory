swfaddressbrowserhistory by iconara

Flex BrowserManager + SWFAddress integration

The Flex 3 framework provides a way to support deep linking and search engine optimization (SEO) in the BrowserManager class. However, it requires you to use the supplied HTML templates, which aren't great. This package makes it possible to use the BrowserManager with applications embedded with SWFObject using SWFAddress as history manager instead.
